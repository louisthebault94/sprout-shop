# Sprout

Educational resource marketplace prototype for primary teachers in Australia and the UK.

Built from the [TeachReach Design System](https://claude.ai/design) handoff bundle.

## Local

Open `index.html` in a browser — no build step. Uses React + Babel-in-browser.

## Deploy

Pushes to `main` auto-deploy to production via Vercel.
- Production: https://sprout-shop-delta.vercel.app
- Project: https://vercel.com/louis-thebaults-projects/sprout-shop
